# kimkim
My Trip Plan Builder

## Getting Started

Follow the steps below to run the server!

### Installing

Install all the required node modules.

In the root directory(kimkim), run npm install

```
npm install
```

In the root directory(kimkim), run client install to install dependencies needed for client

```
npm run client-install
```

If you do not have nodemon, install nodemon

```
npm install nodemon
```


### Connect with Postgres

create .env file in the root directory and add followings:
```
DB_HOST=localhost
DB_USER=(your username)
DB_PASS=(your password)
DB_DATABASE=(your database name)
```

### Get started!

Run dev to start both server and client!
```
npm run dev
```
